# C API - String

Operations:

- `ltn_string_data`
- `ltn_string_size`

## ltn_string_data

`const char * ltn_string_data(ltn_String string)`

Returns the underlying character array of string.

## ltn_string_size

`const char * ltn_string_size(ltn_String string)`

Returns the size of the string in bytes.
