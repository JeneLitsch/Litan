# String
The string type is used to store an array of char values.
A new string can be created by using a string literal e.g. `"Hello World"` or `"Test"`.

**Example**
```js
function main() {
	var a = "Hello";
	var b = "World";

	// print HelloWorld
	std::print(a);
	std::println(b);
}
```


## Escape sequences
`\n` for newline  
`\t` for tab  
`\\` for \  
`\"` for "
