# Array 

The array type is used store a dynamic array of elements of any type.
A new array can be create by writing a comma separated list between `[` and `]` e.g. `[1, 2, 3,]` or `[42, 42]`. 
A single trailing comma is allowed.

## Example
```js
function main() {
	var array = [1, 2, 3,];
	std::println(array);
}
```
