# Char

The char type is an 8-Bit integral type used to represent a single character.
A value of type char can be created by a char literal e.g. `'a'`, `'\n'` or `';'` 

**Example**
```js
function main() {
	var a = 'a';
	var b = 'b';
	var newline = '\n';
	std::print(a);
	std::print(newline);
	std::print(b);
	std::print(newline);
}
```
