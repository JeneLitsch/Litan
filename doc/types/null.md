# Null / std::Null

The null type indicates a non-existing value. A value of type null can be created by the `null` keyword.

**Example**
```js
function main() {
	var a = null;
}
```

## Static Functions

### . new() : Null

Returns `null`.

### . is(value : Any) : Bool

Checks whether value is `null`.

### . cast(value : Any) : Bool

Returns `null`.
