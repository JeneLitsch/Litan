# std::sort_*

## std::sort_ascn(range)
**Parameters:** 
* range: A range or an array

**Return value:**   
null  

**Description:**     
Sorts all elements in range in ascending order.  
The sorting is done in place. This means the provided range is changed.  

---



## std::sort_desc(range)
**Parameters:** 
* range: A range or an array

**Return value:**   
null  

**Description:**     
Sorts all elements in range in descending order.  
The sorting is done in place. This means the provided range is changed.
