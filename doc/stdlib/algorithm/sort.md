# std::sort_*

## std::sort_ascn(range)
### Parameters:
* range: A range or an array

### Return value:  
null  

### Description:  
Sorts all elements in range in ascending order.  

---



## std::sort_desc(range)
### Parameters:
* range: A range or an array

### Return value:  
null  

### Description:  
Sorts all elements in range in descending order.  

---
