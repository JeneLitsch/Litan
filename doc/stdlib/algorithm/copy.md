# std::copy_*

## std::copy_front(range, target)
### Parameters:
* range: A range or an array
* target: An array

### Return value:  
null  

### Description:  
Copies all elements in range front of target.  

---



## std::copy_back(range, target)
### Parameters:
* range: A range or an array
* target: An array

### Return value:  
null  

### Description:  
Copies all elements in range to the back of target.  

---
