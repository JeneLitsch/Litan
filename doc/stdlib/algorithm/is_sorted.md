# std::is_sorted_*

## std::is_sorted_ascn(range)
### Parameters:
* range: A range or an array

### Return value:  
null

### Description:  
Checks if a range or an array is sorted in ascending order.

---



## std::is_sorted_desc(range)
### Parameters:
* range: A range or an array

### Return value:  
null

### Description:  
Checks if a range or an array is sorted in descending order.
