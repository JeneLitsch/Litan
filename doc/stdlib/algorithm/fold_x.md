# std::fold_l / std::fold_r

## std::fold_l(iter, fx, init)
**Parameters:**  
* iter: Iterable object
* fx: Binary function.
* init: Starting value.

**Return value:**  
any

**Description:**   
Sums up all elements using `fx` as a binary operation.

## std::fold_l(iter, fx)
**Parameters:**  
* iter: iterable object
* fx: Binary function used to sub up elements.

**Return value:**  
any

**Description:**   
Sums up all elements using `fx` as a binary operation.


## std::fold_r(iter, fx, init)
**Parameters:**  
* iter: Iterable object
* fx: Binary function.
* init: Starting value.

**Return value:**  
any

**Description:**   
Sums up all elements in reversed order using `fx` as a binary operation.

## std::fold_r(iter, fx)
**Parameters:**  
* iter: iterable object
* fx: Binary function used to sub up elements.

**Return value:**  
any

**Description:**   
Sums up all elements in reversed order using `fx` as a binary operation.
