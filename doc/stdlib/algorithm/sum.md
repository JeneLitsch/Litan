# std::sum

## std::sum(range, start)
**Parameters:** 
* range: A range or an array
* start: Start value to add to

**Return value:**   
The calculated sum  

**Description:**     
Adds all elements to start.  

---



## std::sum(range)
**Parameters:** 
* range: A range or an array

**Return value:**   
The calculated sum  

**Description:**     
Starts with 0 and sums all elements in range.
