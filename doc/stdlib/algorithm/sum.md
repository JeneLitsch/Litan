# std::sum

## std::sum(range, start)
**Parameters:** 
* range: a tuple or array
* start: Start value to add to

**Return value:**   
The calculated sum  

**Description:**     
Adds all elements to start.  

---



## std::sum(range)
**Parameters:** 
* range: a tuple or array

**Return value:**   
The calculated sum  

**Description:**     
Starts with 0 and sums up all elements in the range.
