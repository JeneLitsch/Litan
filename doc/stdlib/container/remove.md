# std::remove_*

## std::remove_front(container)
**Parameters:**  
* container: Array

**Return value:**  
null

**Description:**  
Removes the first element.

---

## std::remove_back(container)
**Parameters:**  
* container: Array

**Return value:**  
null

**Description:**  
Removes the last element.

---

## std::remove(container, i)
**Parameters:**  
* container:
  1. Array
  2. Map
* i:
  1. Index
  2. Map Key 

**Return value:**  
null

**Description:**  
Removes element at index/key.
