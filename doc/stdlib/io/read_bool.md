# std::read_bool

## std::read_bool()
since 1.2  
**Parameters**  

**Return value**  
Bool  

**Description**  
Reads a single boolean from stdin.
0 and false evaluate to false
1 and true evaluate to true

---

## std::read_bool(stream : istream)
since 1.2  
**Parameters**  
* stream: an input stream

**Return value**  
Bool  

**Description**  
Reads a single bool from stream.
0 and false evaluate to false
1 and true evaluate to true
