# std::read_char

## std::read_char()
**Parameters:**  

**Return value:**  
optional&lt;char&gt;  

**Description:**  
Reads a single character from stdin.
Null is returned if an error occurs.

---

## std::read_char(stream : istream)
**Parameters:**  
- stream: an input stream

**Return value:**  
optional&lt;char&gt;  

**Description:**  
Reads a single character from stream.
Null is returned if an error occurs.
