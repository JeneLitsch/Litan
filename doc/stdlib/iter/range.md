# std::range, std::iter::range

## std::iter::range(begin, end, step) const
**Parameters:**
- begin: begin value
- end: end value
- step: step between generated value

**Return value:**  
range&lt;int&gt;

**Description:**  
Creates an range iterator


## std::iter::range(begin, end) const
**Parameters:**
- begin: begin value
- end: end value

**Return value:**  
range&lt;int&gt;

**Description:**  
Creates an range iterator with default step 1
