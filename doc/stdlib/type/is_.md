# std::is_*

## std::is_null(value)
**Parameters:**
- value: A value of any type to test

**Return value:**  
Returns true if **value** is of type **null** or false otherwise.

---

**Parameters:**
- value: A value of any type to test

**Return value:**

## std::is_integral(value)
**Parameters:**
- value: A value of any type to test

**Return value:**

Otherwise false is returned.
