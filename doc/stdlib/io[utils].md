### std::is_eof(stream)
**Parameters**
* stream: stream to check

**Return value**  
Bool  

**Description**  
Returns true if stream passed EOF.  

---

### std::is_good(stream)
since 1.2  
**Parameters**
* stream: stream to check

**Return value**  
Bool

**Description**  
Returns true if no error occured and stream has not passed eof.
