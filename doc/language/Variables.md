# Variables


## Syntax

Create a variable

```js
var /*name*/;
```

or

```js
var /*name*/ = /*expression*/;  
```

In the first case the variable is default assigned to `null`.

## Examples

```js
function main() {
	var foo;
	var bar = 42;
	var baz = "Hello World";
	std::println(foo);
	std::println(bar);
	std::println(baz);
}
```
