# Variables

Variables can be declared inside functions and in global space.

## Syntax

The following code listing shows a the structure of a variable declaration.

```js
var /*name*/ = /*expression*/;  
```

The type and expression is optional.

## Examples

```js
function main() {
	var foo; 
	var bar = 42; 
	var baz = "Hello World"; 
	std::println(foo);
	std::println(bar);
	std::println(baz);
}
```
