# if-else

```js
if(condition) {
    // some code
}
```
or

```js
if(condition) {
    // some code
}
else {
    // some other code
}
```

or

```js
if(condition) {
    // some code
}
else if(condition) {
    // some code
}
else {
    // even more code
}

```

## Note

The condition expression must be convertible to true or false e.g. a boolean.
